### Система управления объявлениями для модерации

## 1. Описание проекта.

Веб-приложение для модерации объявлений на платформе Авито. Упрощённая версия внутренней системы, которую используют
модераторы для проверки и управления объявлениями пользователей.

## 2. Запуск проекта.

git clone https://github.com/ttseva/advertisement-moderation \
cd advertisement-moderation\
npm run start
